# 沐寧「心情數據版」· 顯示方式調研與定案建議（2026-07-02）

> Edward 交辦：每次對話的情緒感知（語氣/用語/用詞）要分析並記錄到 App，成為心情指標數據版；調研「用什麼顯示比較好」。
> 調研範圍：2026 心情追蹤 App（Apple 心情記錄/Daylio/How We Feel/Finch）、穿戴裝置健康分數（Oura/Whoop/Fitbit/Garmin）、長照家人看板（inTouch/ElliQ/CarePredict/Birdie/日本見守り）、長輩資料視覺化研究、華人色彩文化。全程附來源。

---

## 一、定案建議（🥇 主推）

**「天氣式一句話心情 ＋ 3 句觀察 ＋ 暖色週色帶 ＋ 溫柔關心提示」**，子女端／長輩端分開設計：

### 子女端（買的人要 3 秒安心）
- **每通電話一則**：聊天摘要＋心情印象＋話題標籤（inTouch 已驗證此商業骨架可行）。
- **每天聚合一個「今日心情」**：5 級天氣式狀態（不是分數）：
  `很開朗 ☀️ / 不錯 🌤 / 平平 ⛅ / 有點悶 ☁️ / 悶悶的 🌧`
  ——刻意不用「低落/負面/憂鬱」；「有點悶」是台灣日常口語＝觀察感、不是判定感。
- **3 句觀察**（三訊號的顯示形態——是句子、不是分數）：
  - 語氣 →「聲音聽起來」：有精神／平穩／比較累
  - 用語 →「聊天狀態」：話匣子全開／話比平常少
  - 用詞 →「聊了什麼」：話題標籤＋提到開心的事/掛心的事
- **趨勢**：7 日暖色色帶＋30 日月曆色塊（Daylio 熱圖模式、被驗證最會「看出模式」）。
- **溫柔關心提示**：只在「**和她自己平常不一樣**」（連續偏離個人基準線）時出現，文案是「也許是打通電話回家的好時機」——**不是警報**。
- **硬觸發通知只有兩種**（學 inTouch）：連續未接、明確提到身體不適/求助。

### 長輩端（絕不讓長輩覺得被評分）
- **永遠不顯示任何分數或情緒評級**——只有正向回憶卡（「今天我們聊了孫子，你笑得好開心」）。
- Phase 2 可加**花園隱喻**：每天聊天＝澆花、一週開一朵花（Finch 已證明「照顧型隱喻」對脆弱用戶零羞辱感、高黏著；種花剛好是台灣長輩日常）。

## 二、內部運算 vs 顯示的鐵律

```
內部（不顯示）：語氣→連續值｜用語→話量/主動性｜用詞→情緒詞彙傾向 → 加權成每通「心情印象」
顯示（給人看）：離散 5 級狀態＋觀察句＋色帶——永遠是「觀察」、不是「判定」
```
- **絕不顯示**：逐句情感分數、語氣原始值（「音調低沉度 71%」）、0-100 情緒指數、「憂鬱/焦慮風險」字樣。
- **跟自己比、不跟量表比**（CarePredict「not their normal self」模式）：媽媽本來就話少就不該天天觸發「話少」——這同時是最強的非醫療化設計。

## 三、最重要的反面教訓
- **Amazon Halo Tone**：給使用者的「聲音語氣」打分（正向度/能量）→ 被媒體與 Mozilla 罵爆「史上最私密資料、最毛的麥克風」→ 2023 全線下架。**沐寧啟示：我們照樣做語氣分析（內部訊號），但顯示層永遠說「聊得很開心」、永遠不說「語氣正向度 68%」。**
- 圓環/分數（Oura 式「媽媽今日心情 62 分」）＝焦慮製造機＋醫療儀表味 → 否決。
- 折線圖情緒指數＝把情緒當生命徵象在量 → 否決（原始線圖）。
- 用紅色標壞心情日：華人紅=喜慶、且紅=警報會把「悶」框成「出事」→ 色帶用暖橘金=開朗、淺灰藍/灰=較悶，紅只留給喜慶。
- 每天例行推播負面狀態 → 事件驅動才對（ElliQ/日本見守り驗證：安心不需要高頻高粒度）。

## 四、學理與市場依據（精選）
- 離散情緒標籤比連續維度更易被一般人理解（Nature Sci Reports 2025）。
- 長輩視覺化研究（JMIR 2023/GerontoVis）：大字、少色、聚合總覽勝過細粒度。
- 天氣隱喻：自帶「會過去」的去災難化暗示、中文語感天然（心情陰天/雨過天晴）；但作為家人看板主視圖尚無 shipped 標竿＝我們要自己驗證（誠實聲明）。
- 已上市可抄骨架：inTouch（per-call 摘要+心情+話題+家人通知）、ElliQ（事件驅動警示）、Apple 心情記錄（色彩系統）、Daylio（色塊月曆）、Birdie（探訪後家屬觀察報告）。
- 三訊號（語氣/用語/用詞）加權無公開文獻 → 上線後拿頭兩個月真實通話校準（誠實聲明）。

## 五、與現有架構的接法
- 資料來源＝感知定案規劃 §5 的 P1 情緒模組：聊完 → 三角度判讀 → `WellbeingSignal`（帶信心度）→ 時序趨勢庫（個人基準線）。
- 數據版＝讀趨勢庫的顯示層：每通印象 → 每日聚合 → 週/月色帶；「偏離基準」規則產生溫柔提示。
- 隱私沿用信任定案：家人看摘要與觀察、不看逐字稿/錄音；長輩知情且可限制。
