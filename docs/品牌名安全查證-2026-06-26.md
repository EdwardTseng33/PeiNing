## 1. 網址能不能拿 — 🟡（要用的兩個都可拿、.com 被佔不影響）

| 域名 | 燈號 | 狀態 | 證據（實查） |
|---|---|---|---|
| **peining.app** | 🟢 可拿 | App 天然主網址、空的 | Google DoH 回 NXDOMAIN + RDAP 三端點全回 404「not found」+ DNS 無 NS 記錄。實查 |
| **peining.com.tw** | 🟢 可拿 | 台灣主戰場、空的 | TWNIC 官方查詢回「未註冊」。實查 |
| **peining.tw** | 🟢 可拿 | 台灣短後綴、空的 | TWNIC 官方查詢回「未註冊」。實查 |
| **peining.com** | 🔴 被佔 | 1999 註冊、GoDaddy 名下、續到 2031 | Verisign RDAP 完整紀錄。實查 |

打法（主網址走 `.app` + 台灣走 `.com.tw`）在陪寧身上完全成立。`.com` 被佔是 2026 常態（短英文 .com 早掃光），代價只是品牌防禦缺一角；持有者身分被遮蔽、未明（若想評估收購再單獨查 GoDaddy 頁面）。**不擋立項。**

## 2. 撞名 / 搜尋競爭 — 🟢（藍海）

銀髮 / 長照 / 照護領域**零**同名品牌、公司、產品。近音只有 PEIPING 精密（1980 磨床廠、領域差到外太空）+ 一個人名（Peining Chang，阿茲海默協會員工）——皆非品牌、零混淆。Google 搜尋第一頁可自己佔。

## 3. App Store / Google Play 同名 — 🟢

無任何 App 叫「陪寧」或「PeiNing」。同類 App 是別的名字（銀髮健康、仁寶 i 照護、家天使、恬心陪伴…）。上架不撞名。

## 4. 台灣商標（TIPO 官方檢索 · 資料更新到 2026-06-25）— 🟢

- **核心三類別撞名數 = 0**：第 9 類（軟體 / App）0、第 42 類（軟體設計）0、第 38 類（通訊）0。
- 無精確「陪寧」二字商標。近似的 35 筆九成集中在第 5 類藥品（倍特寧、康倍寧…藥品命名邏輯），含字串的只有「舒陪寧」「康世德美陪伴寧」兩筆、皆第 5 類藥品、不同領域不衝突。
- 英文「PEINING」全台僅 1 筆（PEI NiNG 及圖、第 3 類化妝品、1997 註冊**已到期消滅**）、無障礙。
- 正面訊號：同類競品「恬心陪伴」（Wicare 慧利眾）已在跑 = 市場驗證，名字不撞。

## 5. Apple 命名規範 — 🟢（名字合規 + 一條操作紅線）

「陪寧 / PeiNing」本身完全合規、Apple 無針對性限制。**但健康類 App 紅線**（跟 Edward 既定法規線一致）：
- App 名稱 / 副標 / 關鍵字 / 截圖文案全程避開「醫療 / 診斷 / 治療 / 藥師 / 諮商」字眼。
- 用藥提醒功能描述成「**提醒服藥時間**」、**不要碰「劑量 / 給藥建議」**（Apple 對藥物劑量計算有嚴格限制、會誤判成醫療器材）。

## 6. ⭐「安寧」聯想風險 — 🟡（兩輪分歧、原樣攤開）

兩輪獨立判斷都給 🟡、都同意硬指標全綠、都同意「陪」字把詞義拉回「陪伴」（招牌字是「安/療/緩和」不是單一「寧」）；**差別在長輩端的嚴重度**：

- **輪一（偏可控）**：長輩端**低**、子女端中。結論「可直接用，把文案 / 主視覺定錨在『安心、平靜』就好」，Gate 3 盯死「寧 = 安心」不滑向臨終側。
- **輪二（偏謹慎）**：長輩端**較高**、子女端中。理由——台灣長輩對「死亡」高度忌諱（安寧療護機構自己的觀察），「陪伴 + 高齡 + 寧」三訊號疊起來剛好是安寧病房場景。結論「立項繼續、名字先掛『暫定』，**在女巫大規模套進 UI 前先讓蕪菁頭跑一輪 60+ 長輩 + 子女的『第一聯想』小測**」。

**收口建議（蘇菲）**：兩版分歧的點正是「長輩會不會往臨終想」——這種事**在城堡裡推演不如去問真長輩**（品牌不是我們說它是什麼、是用戶說它是什麼）。半天小測（5+ 長輩、5+ 子女）即可定案：
- 測過關 → 正式鎖名、文案定錨「安心」往下衝。
- 測亮黃燈 → 備選彈藥（保氣質、去陰影）：**陪安 / 陪暖**（保「陪」換「寧」）、**康寧 / 寧安**（保「寧」但拉向「健康」）、或回 6/26 原推**暖陪 Nuande**。

## 7. 跨區商標粗掃（WIPO 全球庫）— 🟡（只關未來海外）

全球「PEINING」20 筆全是英文 PEINING + 圖形 logo、3 組持有人，**無一是中文「陪寧」、無一是銀髮 / 健康 App**：
- 美國 PEINING（深圳創沛匯、電子）**狀態 Ended 已失效** → 美國當下無有效障礙。
- 中國深圳那家在歐盟 / 中南美持 class 9 PEINING → **僅當日後用英文 PEINING 當主品牌進歐盟 / 中國海外**才可能撞；我們主品牌是中文「陪寧」、英文只是副標、且 logo 不同 → 衝突面很窄。
