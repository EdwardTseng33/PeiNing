## 2. 核心定義（一眼看懂沐寧）
- **官方一句話定位（Edward 6/28 鎖定）**：「**會記得你，也在乎你在乎的人。**」靈魂＝懂得何時該講、何時該靜，讓人驚豔的感知與聊天互動。
- **是什麼**：住手機裡、會說話、記得你、出事會找人的「家人替身」。**子女買、長輩用**。智慧健康／照護、**非醫療軟體**。
- **核心畫面**：點開 → **全屏管家臉、會說話、會動、超大結束鈕**（像跟家人視訊）。畫面**像視訊通話、不放逐字稿/字幕當主畫面**；只顯示必要狀態（在聽／在說）與安全提示。字幕僅列為**未來給重聽長輩的可選輔助（預設關）**。→ 依 2026-06-29 S2S 產品框架校正（見 `STATUS.md`／`README.md`／`CURRENT-DEVELOPMENT-PLAN.md`），**取代舊「即時字幕為核心元素」的寫法**；SPEC §5/§6 仍有舊字幕寫法待同步（見 §4）。
- **三顆腦＋角色人格層＋一張臉**：反射腦(即時語音)＋管家腦(背景感知/記憶)＋守護腦(危機安全網)＋角色人格層(六角色、使用者命名、語氣/關係/表達)＋一張臉(Ditto/2D 會動的臉)。
- **真正回話公式**：`回應 = 角色人格 + 使用者記憶 + 即時感知 + 當下對話 + 安全規則 + 語音表達限制`。
- **AI 模型現況（7/1 更新）**：反射腦（即時語音）方向 = Gemini 3.1 Flash Live（待正式串接）；**管家腦（Butler）與守護腦（Guardian）已改用 `Claude Sonnet 5`**（`claude-sonnet-5`，見 `engine/model_router.py`）。SPEC §4.3 仍寫「Gemini Flash-lite 級·建議待卡西法成本細算」為舊建議，實際 code 已定案為 Sonnet 5，以此頁與 code 為準。
- **付費模式現況（7/1 更新）**：**訂閱制 `Free / Plus / Premium / Concierge`**（非中文四檔命名）。訂閱＝可預期的基礎陪伴體驗（不因聊天扣點）；點數只保留給貴的加值（premium Avatar/GPU 超額分鐘）。權威見 `BILLING-CREDITS-ENTITLEMENT-v1.md`。
