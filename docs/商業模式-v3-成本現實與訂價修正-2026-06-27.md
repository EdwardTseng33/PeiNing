## 一、語音腦真實價（查證 3 來源：tokencost / callsphere / OpenAI 官方）
**gpt-realtime-2（2026-05-07 GA、目前最新生產版）**：audio input $32 / 1M tokens（cached $0.40、省 98.75%）、audio output $64 / 1M tokens。約舊版 gpt-4o-realtime（in $100/out $200）的 1/3 價。
- **無 mini 版 realtime**（OpenAI 2026-05-07 只推 gpt-realtime-2 / translate / whisper）→ **省錢沒有「換便宜模型」這條路、只能靠優化。**
- token 換算：input 1 token/100ms = 600 tokens/分；output 1 token/50ms = 1,200 tokens/分。

**每分鐘語音腦成本（1 USD=32 NTD、假設長輩:寧寧說話 60:40）**：
| 情境 | NT$/分 |
|---|---|
| 基本 uncached | **NT$1.6** |
| 有效 caching（生產最佳化） | **NT$1.0** |
| 最壞（長輩話多、寧寧不停回） | NT$3.07 |

## 二、每分鐘聊天全包成本（動態臉模式）
> 架構前提：臉(Ditto)走 idle-loop、只「寧寧開口段（~40%）」燒顯卡；語音腦「整段都跑」。
```
每分鐘 = Ditto(40% × NT$0.37) + 語音腦(100% × NT$1.5) ≈ NT$1.65/分（uncached）
                                              ≈ NT$1.15/分（caching 後）
```

## 五、Verdict
- **基礎 499 撐不住中度以上用戶**（中度語音腦就燒 NT$675 > 實收 424）。**「無限聊到飽 + 499」會流血。**
- **高級 999「兩位各自無限」風險高**（雙倍變動成本）→ 改「2 長輩**共用一個額度池**」。
- **尊榮 1999 在「有額度 + 語音輕中度」健康**；語音也要日封頂（~30-40 分）防話多者吃光 120 分溢價。
- **無 mini 可換** → 省錢三招：① prompt caching 全開（input 省 96%、效果最大）② VAD 靜默不計費（長輩超會停、省 20-30%）③ 控 context 長度。
- 自架語音腦：v1 先 caching；Year2 用戶 5,000+ 再評估（gpt-realtime-2 的口音理解+情緒語氣難自架複製）。

## 六、🎯 救 NT$499（待 Edward 拍）
- **Road A（💡 CFO 強烈建議）**：「聊到飽」改「**每天大方額度、睡醒滿格**」（基礎 ~15 分/天、按檔遞增）+ 強制 caching+VAD（腦價壓 ~NT$0.8）+ **維持原價 499/999/1999**。多數長輩到不了線、重度自己升級/加購、價格有競爭力（錨影音月費 300-460、ElliQ 960-1280）。
