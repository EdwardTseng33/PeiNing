## ⭐ 一鍵重跑（PyTorch 路、已驗證可跑、繞開 tensorrt）
RunPod 範本 `Runpod Pytorch 2.8.0`（CUDA 12.8）· RTX 4090 · Jupyter Terminal 貼：
```bash
cd /workspace
git clone https://github.com/antgroup/ditto-talkinghead && cd ditto-talkinghead
apt-get update -y && apt-get install -y git-lfs ffmpeg libgl1 libgles2 libegl1 libglib2.0-0 libxrender1 libsm6 libxext6 && git lfs install
pip install librosa tqdm filetype imageio opencv_python_headless scikit-image cython cuda-python imageio-ffmpeg colored polygraphy "numpy==2.0.1" onnxruntime mediapipe einops
git clone https://huggingface.co/digital-avatar/ditto-talkinghead checkpoints
# 若 checkpoints 只下載到指標檔(很小)：cd checkpoints && git lfs pull && cd ..
time python inference.py \
  --data_root "./checkpoints/ditto_pytorch" \
  --cfg_pkl "./checkpoints/ditto_cfg/v0.4_hubert_cfg_pytorch.pkl" \
  --audio_path "./example/audio.wav" --source_path "./example/image.png" \
  --output_path "./tmp/result.mp4"
```
**踩過的坑（已解、留紀錄）**：
1. `tensorrt==8.6.1` 在 CUDA 12.8 編不過 → **改用 PyTorch checkpoint**（`ditto_pytorch` + `v0.4_hubert_cfg_pytorch.pkl`），不裝 tensorrt。
2. `onnxruntime-gpu` 要 CUDA 13（機器是 12）→ **改裝 CPU 版 `onnxruntime`**（臉部偵測在 CPU、不影響主算圖速度）。
3. 缺 `mediapipe`、`einops` → 補裝。
4. mediapipe 缺系統圖形庫 `libGLESv2` → `apt-get install libgl1 libgles2 libegl1 …`。
