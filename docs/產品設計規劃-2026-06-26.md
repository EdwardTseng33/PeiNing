## 1. 市場機會（霍爾深度調研）

**值得做——正在裂開的藍海縫，大到不會錯過，但還沒有贏家站穩，台灣 C 端是空的。**

- 全球 AI 陪伴 App：US$50 億（2026）→ US$96 億（2030）、18% 年成長。
- 「社交陪伴」是 AI 銀髮照護**成長最快子類**（19.7%）——市場正從「監測身體」往「陪伴情緒」傾斜，正好我們的主軸。
- 台灣 65+ = 467 萬（占 20%）、獨居近 100 萬。
- **台灣 C 端沒贏家**：強者全是 B2B（智齡科技做機構 SaaS）。C 端只有零星弱玩家。窗口 12-18 個月。
- ⚠️ 誠實：單吃台灣會餓（台灣軟體 AI 陪伴 2035 才估 30 億），**真正的量在輸出海外**。

## 2. 目標用戶（蕪菁頭深挖）

**核心真相：長輩不會主動開 App。這不是設計問題、是人的問題。**

- **雙主角，缺一不可**：Tier 3 子女（35-55，下載/付費/裝機/看儀表板）+ Tier 1 長輩（70-85，被服務、不主動操作）。**行銷打子女、產品圍繞長輩，兩個邏輯同時成立。**
- **次甜蜜點**：Tier 2 預防型（60-70，會自己用、自己付，介面別有「銀髮/照護」味）。
- **長輩用 App 的死穴**：① 首次安裝靠長輩自己=死（必子女代裝）② 依賴長輩記得開 App=日活爛掉（**主動出聲是命脈**）③ 引導超過三步=退出 ④ 子女裝機沒調好手機通知/音量=Aria 喊不到。

## 3. 定位與差異化

**砍掉假差異**（這些是入場券、不是護城河）：AI 語音對話（人人有）、用藥提醒（Medisafe 4.7 分輾壓）、情感陪伴（Replika 3000 萬用戶）。

**три 真差異**：
1. **純 App + 台語/華語 + 隨身** — 繞開硬體對手三重死穴（ElliQ/Hyodol/Sesame 全綁硬體 + 英/韓語，轉純 App 等於砍自己金流；台語是在地化硬門檻、戳中「國語太正式」）。
2. **一張會記事的臉 + 主動開口** — 比語音音箱多情感錨點、比娃娃硬體少成本。長者要的是「記得我、會主動關心我」。
3. **危機安全網 + 家人連動** — 用 character.ai 的血換來的定位（青少年自殺訴訟）。別人當 v2 補丁，**我們當 P0**。子女買的就是「我不在時有張安全網」。

## 4. ⭐ Careon 原規劃 Review（Edward 指定：哪些改 / 強化）

Careon 原本是**軟硬體大方案**（Home Pad 機器 + 感測器 + 雲端 + App）。純 App 化的取捨：

### 🔴 砍掉（要硬體、純 App 做不到）
- CAREON Home Pad（7" 硬體 hub）、衛星感測器（浴室/廚房/臥室）、端側運算盒
- 硬體感測：PPG 心率 / PIR 動作 / 環境溫濕 / **跌倒偵測**（硬體版）
- CGM 血糖整合、視訊鏡頭心理感測（隱私風險高、v1 不碰）

### 🟢 保留（軟體層、直接做）
- **2 個角色 Aiden / Aria**（核心人設、原封不動）
- **用藥提醒**（但收斂成「純鬧鐘 + 語音念」，守法規線）
- 語音情緒感測（端側）、主動陪聊、心理衛教（冥想/呼吸/老歌懷舊）、家庭聯繫
- 5 Tier 用戶框架、定價骨架（NT$499-1299）

### 🟡 強化（調研發現必須升級的）
| 從 Careon 的… | 升級成… | 為什麼 |
|---|---|---|
| 「主動陪聊」一個功能 | **主動觸發 = 核心架構** | 蕪菁頭：長輩不主動開 App、主動是命脈 |
| 「專業諮商導入」附屬 | **危機安全網 = P0** | 霍爾：character.ai 教訓、子女買的是安全網 |
| 「Family App」一端 | **家人儀表板 = 子女付費核心動機** | 蕪菁頭：長輩使用量綁子女續訂 |
| 台語「可選」 | **台語 = 招牌差異** | 霍爾差異化 + 蕪菁頭「國語太正式」 |
| 記憶「健康輪廓」 | **記憶連續性**（記得上次說的話） | 蕪菁頭：黏著關鍵、Day 30 留存 |

### ➕ 新增（Careon 沒有、調研挖出來的）
- **子女代裝 onboarding 檢查清單**（含手機通知/音量/螢幕設定調校——蕪菁頭死穴 4）
- **雙介面**：子女版（設定/儀表板/提醒管理）+ 長輩版（一個「和 Aria 說話」入口、第一畫面就是臉）
- **14 天免費試用**（蕪菁頭：Tier 3 轉換必要）
- **Aria 第一句話設計**（決定性 bet）
- **Apple Watch / HealthKit 選配整合**（純 App 補跌倒偵測的唯一路）

## 5. 產品功能架構（雙介面）

```
長輩版（極簡 · 語音為主）          子女版（管理 · 資訊豐富）
┌─────────────────────┐         ┌─────────────────────┐
│  Aria 的臉（會說話）  │         │  家人儀表板          │
│  「你好，我是 Aria」  │         │  · 今天聊了嗎/情緒    │
│   點一下就講          │         │  · 藥吃了嗎          │
│  （主動出聲找長輩）   │         │  · 安全網警示        │
└─────────────────────┘         │  用藥提醒設定        │
  大字 24px+ / 按鈕 60px+         │  代裝 onboarding     │
  只有一個主按鈕                  └─────────────────────┘
```

底層（卡西法）：BeyondVoice 引擎七成複用——即時語音 + 會說話的臉 + 深想腦 + 記憶 + 登入現成；用 Capacitor 把網頁裝進 iPhone 殼，語音 + 推播用原生補。

## 6. 第一版範圍（MVP · 全在法規紅線內）

1. **語音陪聊**（Aria/Aiden·台語+華語·記得上次說的話·主動出聲）= 核心
2. **用藥提醒**（純鬧鐘：子女設好 → 到點 Aria 語音念 → 紀錄吃沒吃 → 漏服通知子女）**不建議任何藥/劑量**
3. **情緒陪伴 + 危機安全網**（感知情緒變化 → 偵測危機關鍵詞 → 通知家人 / 接 1925 / 119）AI 是轉介者不是治療者
4. **家人儀表板**（子女看：聊了沒/藥吃了沒/情緒摘要）
5. **子女代裝流程**（15 分鐘裝好設好 + 手機設定檢查清單）

**第一版明確不做**：用藥建議/劑量/衛教、任何診斷、心理治療/諮商定位、硬體感測、跌倒偵測（除非接 Apple Watch）、視訊心理感測。

## 7. 用戶旅程關鍵設計

- **廣告**：打子女焦慮非長輩功能（「Aria 24h 陪你媽說話、吃藥了她告訴你」）
- **落地頁第一屏**：必須回答「媽媽真的會用嗎」（最高流失點）+ 30 秒影片展示長輩只要說話 + 14 天免費
- **代裝**：≤ 5 步、15 分鐘，含手機設定調校檢查清單
- **Day 1**：Aria 主動出聲 + 第一句話（台語）= 留存決定性因素
- **Day 8-30**：記憶連續性（「上次你說膝蓋痛，好點沒」）跨過 Day 30 大坎
- **持續**：子女儀表板有內容 = 續訂；空白 = 退訂

## 8. 設計原則（銀髮 UX 鐵律）
- 主動找長輩、不等長輩找（命脈）· 語音優先、打字是障礙 · 大字高對比超大鈕 · 長輩端零設定（子女代管）· 第一畫面就是 Aria 的臉、點一下就講 · 不出現「銀髮/老人/照護」字 · 角色是「朋友」不是「照護者」

---

## 9. 商業模式

- **鐵律**：規模在 B2B2C（政府/保險給付），不在純 C 端訂閱（ElliQ/Hyodol/Medisafe 都證明）。
- **打法**：先 B2C 上 Apple Store 練市場（Tier 3 子女買單、NT$499 入門 / 999 / 1299 加值）+ **同步啟動 B2B2C 通路布局**（談判週期 12-18 月、現在不開始就慢了）。
- ⚠️ 台灣長照 3.0 補助 2026/7 上路但**純軟體還沒進給付**——B2B2C 別當 Year 1 既定收入。

---

## 10. 風險與紅線
- **法規（沙利曼）**：用藥=鬧鐘不是藥師、情緒陪伴不稱諮商治療、失智長輩同意能力是最硬坑。
- **Apple 審核（卡西法）**：用藥那條線守住「鬧鐘≠藥師」+ 文案零醫療宣稱 → 過審機率高。
- **character.ai 教訓**：危機安全網是 P0 不是補丁。AI 失控內容要護欄。
- **競爭**：窗口 12-18 月、巨頭語音在逼近——速度是關鍵。
