## 0. 一個改變方向的策略事實 ⭐

**短的、乾淨的 4-5 字母英文 .com 全沒了**（2026 鐵律、好唸的 20 年前就被掃光）。逐一實查 yolu / anho / aria / hearo / nuan / heyaria.com 全 TAKEN。

→ **正確打法**：取一個夠獨特的名 → 主網址走 **`.app`**（Apple Store 產品的天然網址、新、幾乎一定拿得到）+ 台灣走 **`.com.tw`**；`.com` 能加字拿就拿、拿不到不強求。

## 1. 最推薦

### 🥇 暖陪 / Nuande（中文掛帥）— 蘇菲 + 霍爾都最推

- 「**暖**」= 陪伴的體溫，「**陪**」= 這 App 在做的事。兩個字直擊靈魂。
- **76 歲阿嬤唸得出、45 歲女兒看得懂**——完全打中「用的人 ≠ 付錢的人」。
- 零撞名（**刻意避開「暖暖」**——閃耀暖暖手遊太紅、含「暖暖」SEO 會被蓋死）。
- 不碰醫療紅線（沒有醫/療/藥/care/med），沙利曼那關天然過。
- 英文 Nuande = 拼音生造詞、英文圈搜尋幾乎空白（藍海）。`.app` / `.com.tw` 判斷可拿。
- **唯一 trade-off**：Nuande 對英語母語者不直覺（要學著唸）。但主市場台灣 + 中文掛帥，值得。
- **蘇菲判斷**：我最推這個。銀髮產品，「阿嬤記得住」比「英文漂亮」重要一百倍。

### 🥈 AidenAria / Anaria（英文品牌即故事）— 策略備選

- 從兩個 AI 家人角色（**Aiden + Aria**）長出品牌名——產品本身就是品牌故事。
- **aidenaria.com 實查 = AVAILABLE** ✅（少數還空著的乾淨 .com）。
- Anaria（縮合）更短、唸起來像義大利文、有溫度、零撞名。
- 缺點：中文要另取（建議搭「暖陪」當中文）、偏英文感。
- **蘇菲判斷**：很有靈魂，但對台灣銀髮市場英文有距離感。當英文品牌名 / 國際化備選。

## 4. 查證方式（2026-06-26 · Edward 最在意的）

- **撞名 / 搜尋競爭**：WebSearch 逐名搜（含 App Store / Google Play / 品牌）
- **域名可註冊性**：RDAP 官方查詢協定（比 Namecheap 網頁準、後者被擋）
  - 實查 TAKEN：yolu / anho / aria / heyaria / nuannuan.com

- **商標初篩**：TIPO（台灣智慧財產局）+ Apple App Store 同名比對
- **⚠️ Caveat（誠實）**：`.app` / `.com.tw` 精確狀態這輪是推斷（RDAP 的 .app 端點連線被擋）。**蘇菲會在 Edward 挑定方向後逐筆官方確認**（或派卡西法直接開註冊商頁面截圖），再交沙利曼跑商標 + Apple 命名規範終審。
