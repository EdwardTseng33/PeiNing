## 0. 一句話解決方案
給每位長輩配一個**只屬於他、30-35 歲、記得他、感覺得到他、出事第一個察覺**的專屬管家——**整套自架、對話像真人、深度懂他卻不拖慢**。

## 1. 定位（鎖定）
- **核心族群**：長輩 60+（被服務的主人）。**買單**：子女。
- **形象（衣服）**：30-35 歲專屬管家（Aria 女 / Aiden 男，既有設定延用），溫暖、能幹、不擺架子、**去老人味**。
- **本質**：個人 AI 專屬管家——主動、懂你、替你張羅、照看你。不是聊天玩具、不是被動陪伴。
- **引擎可複用**：管家腦蓋在現有語音引擎（castle-voice-engine）上；同一引擎日後能「穿別的衣服」服務其他族群。
- **語言（前期）**：台灣中文 + 台語 + 英語（吃不消則收 中文 + 台語）。台語=招牌一定留、中文幾乎隨台語免費、英語最輕（模型支援最好、非負擔來源）；核心長輩用不到英語、其價值在海外輸出 → 前期主磨中文+台語的長輩體驗、英語先做到「能用」。

## 2. 系統骨架：三顆腦 + 一張臉

### 🧠 反射腦（對話當下 · 要快）
- 即時聽與講、自然你來我往（turn-taking）。
- **被打斷的自然處理**：聽到 → 判斷要不要停 → 停了也可能補完半句 → 接話記得上文，不硬切。
- 低延遲是命脈（長輩等不了）。靠即時語音模型（GPT-Realtime-2 已接）+ 一層「對話節奏」設計。

### 🧠 管家腦（背景 · 深想 · 不擋路）= 管家的靈魂
- **感知層**（每天主動長見識）：清晨先備好「今天」——天氣（連動健康提醒：寒流→添衣量血壓）、回診/用藥/重要日子（生日/節日/忌日）、挑過的在地新聞、子女願意分享的家人動態。
- **記憶層**（深度認識一個人）：喜好/人格/家人成員與名字/情緒/身體/作息規律/生命故事與回憶/禁忌話題/信仰/慢性病史。
- **關鍵**：全在背景跑（清晨、空檔、夜間），對話當下只「取用」已備好的 → **深而不慢**。

### 🧠 守護腦（安全網 · P0）
- 偵測情緒/作息異常 + 危機關鍵詞 → 通知家人 / 轉介 1925 / 119。AI 是轉介者、不自己扛（法規線）。

### 🙂 一張臉（兩種、全自架）
- 30-35 歲管家形象。長輩二選一：① **擬真真人 = 提供 4 個形象（2 男 2 女、類競品）** ② 2D 可愛角色（一組）。
- **關鍵**：4 個臉只是「外表 + 聲音」不同，**背後管家腦（感知 + 記憶）是同一顆、共用** → 多給形象不增加「懂你」的複雜度。
- **成本**：自架成本算「同時幾路串流」、不是「幾種臉可選」→ 多給形象幾乎不增加跑起來的錢，只增加「各做一張臉」的一次性工。選擇可大方給。
- **聲音對外（要）**；**攝影機即時對內認知（第一版不做：貴 + 拖慢 + 隱私）**。要做健康視覺判斷改「偶爾拍一下背景處理」。

## 2.5 互動模型：推播層（平常）+ 對話層（App 開）⭐ Edward 6/26 拍板
- **App 沒開 → 推播層**：標準 iOS 推播（一定到、零 iOS 限制風險）。可帶管家的臉、快速鈕（「吃了」／「晚點」不開 App 就回）、緊急可強制響（突破靜音）、短語音片段。日常問候／用藥／提醒／安全警示走這層、24h 都在。
- **App 打開 → 對話層**：完整即時語音 + 會說話的臉 + 你來我往陪聊（所有「像真人」的功夫在這裡）。
- ✅ **解掉最大技術死穴**：不再依賴「App 沒開也能主動出聲」（iOS 不准）；平常靠推播、深互動在 App 裡。
- 💰 **成本再省**：會說話的臉只在 App 開著時才跑、非 24h 掛著 → 自架機器成本進一步下降（usage 假設要回饋給 Sophie 財務模型）。
- ⚠️ **死穴位移**：從「技術（能不能出聲）」→「行為（長輩會不會點推播、把 App 打開）」。門檻較低（推播主動跳出 + 可做得像家人在叫），但這是新的「第一要驗」行為指標。

## 3. 「很懂你」為什麼不會「很慢」（解 Edward 延遲擔憂）
深功夫（感知 + 記憶）全在**管家腦背景做好**，對話當下**反射腦只取用、不現算** → 再懂你也不卡。
唯一真的會拖慢、又踩隱私的是「攝影機即時盯著長輩邊看邊判斷」→ 所以第一版直接不放進即時對話路徑。

## 4. 全自架 = 商業模式的根
- 整條鏈自架、零第三方每分鐘費。租第三方臉每分鐘 $0.1-0.2 = 每人每月 $30（約 NT$900）光一張臉 → 毛利必死。
- 自架攤平到接近零 → Sophie 財務（每用戶月成本 ~NT$150、毛利 65%）才站得住。
- 技術可行性：2D 軌現可建（Audio2Face + Live2D）；擬真軌 Ditto 單卡待 PoC。詳卡西法自架架構檔。

## 5. 第一版範圍（靈魂對了、範圍要框）
- **必做**：自然對話（反射腦）+ 管家腦「感知 + 記憶」核心 + 守護腦安全網 + 一張臉（先 2D 保底、擬真 PoC 後評估）+ 用藥提醒 + 家人儀表板 + 子女裝機。
- **先不做**：攝影機即時認知、滿配感知（先做天氣/回診/重要日子幾項最有感的）。
